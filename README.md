# ✨ Overview

AshrafRadif Store is a modern fullstack ecommerce platform focused on delivering a premium shopping experience for smartphone products including iPhone, Samsung devices and accessories.

The project was designed with a strong focus on:

- Clean modern interface
- Responsive design
- Admin dashboard management
- Product variant handling
- Performance optimization
- Scalable architecture
- Real-world ecommerce workflows
