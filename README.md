# E-Commerce-Platform-AchrafRadif-
Developed a modern e-commerce platform for smartphone sales featuring a responsive storefront and admin dashboard using Next.js, TypeScript, PostgreSQL, and Tailwind CSS. Implemented product management, authentication, order workflows, and dynamic product config. Integrated Supabase for backend services and PostgreSQL db, Cloudinary for media

## ✨ Overview

AshrafRadif Store is a modern fullstack ecommerce platform focused on delivering a premium shopping experience for smartphone products including iPhone, Samsung devices and accessories.

The project was designed with a strong focus on:

- Clean modern interface
- Responsive design
- Admin dashboard management
- Product variant handling
- Performance optimization
- Scalable architecture
- Real-world ecommerce workflows
